User Retention helps you understand how many users come back after their first activity date.

Use this page to measure returning users, compare retention by date, and understand whether users continue using your app or website after they become active.

## User Retention overview

User Retention shows how users return over time after their first activity date. Each row represents a date-based user group, and each day column shows the return percentage for that group.
