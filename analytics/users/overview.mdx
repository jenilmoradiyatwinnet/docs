User Overview helps you inspect individual users, understand their device details, review activity, and analyze how they move through your app.

Use this page when you want to debug a user journey, check device properties, verify events, or understand where a user came from.

## User flow

<Steps>
  <Step title="Open Users">
    Go to **Users** from the left sidebar to view the tracked user list.
  </Step>

  <Step title="Filter users">
    Use date range, country filter, and search to find the right user.
  </Step>

  <Step title="Open user overview">
    Click a user row to open the detailed user overview page.
  </Step>

  <Step title="Review activity">
    Check total screens, total time, app background, and date-wise session rows.
  </Step>

  <Step title="Inspect properties">
    Review device properties, event properties, traffic source, and screen journey.
  </Step>
</Steps>
