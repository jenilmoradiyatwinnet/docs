User Overview helps you inspect individual users, review their activity, check device details, and understand how they move through your app or website.

Use this page when you want to debug a user journey, verify tracked events, understand traffic source, or analyze individual user behavior.

---

## Page preview

<Steps>
  <Step title="Open Users list">
    The Users list gives a quick overview of every tracked user in the selected project. You can filter users by date, country, or search keyword. ## Example web event

This is an example of a `pageView` event. It shows which page was opened, where the user came from, and which platform sent the event.

```json
{
  "eventName": "pageView",
  "eventType": "auto",
  "appInfo": {
    "appVersion": "1.0",
    "platform": "javascript"
  },
  "properties": {
    "pageCount": 17,
    "pageDomain": "localhost",
    "pageUrl": "http://localhost:3030/tools",
    "pagePath": "/tools",
    "pageTitle": "FF Tool",
    "referrerUrl": "http://localhost:3030/",
    "referrerType": "internal"
  }
}
```

<Info>
  In User Overview, this type of event can be inspected from Event Properties. For web tracking, fields like page URL, page path, page title, and referrer help explain how the user reached that page.
</Info>

## User flow

  <Step title="Open Users">
