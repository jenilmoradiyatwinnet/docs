## Example questions

You can ask questions like:

- Which users were active today?
- Which events are growing this week?
- What is my user retention?
- Which screen has the highest drop-off?
- How many users completed checkout?
- Which campaign performed best?

<Tip>
  Ask AI works best when your project has enough event and user data.
</Tip>

---

## Best practices

<AccordionGroup>
  <Accordion title="Ask clear questions">
    Use simple and direct questions. For example, ask “How many users opened the app today?” instead of vague questions.
  </Accordion>

  <Accordion title="Use date ranges">
    Add time periods like today, yesterday, last 7 days, or last month to get more accurate answers.
  </Accordion>

  <Accordion title="Use event names">
    If you know the event name, include it in your question. For example, “How many users triggered add_to_cart last week?”
  </Accordion>
</AccordionGroup>
