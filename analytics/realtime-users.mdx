## Realtime widgets explained

<AccordionGroup>
  <Accordion title="Active users in last 30 minutes" icon="users">
    This metric shows how many users were active in the recent 30-minute window.

    It is useful when you want to check live traffic after a campaign, push notification, app release, or marketing activity.
  </Accordion>

  <Accordion title="Users by Country" icon="globe">
    This widget shows country-wise active users.

    It helps you understand which regions are currently using your app or website.
  </Accordion>

  <Accordion title="App Open Frequency" icon="chart-line">
    This widget helps you understand how frequently users open the app.

    It is useful for measuring repeated usage and live engagement behavior.
  </Accordion>

  <Accordion title="App Version" icon="code">
    This widget shows app version usage.

    It helps product and development teams understand which versions are active and whether users are moving to the latest version.
  </Accordion>
</AccordionGroup>
