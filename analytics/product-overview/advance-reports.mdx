Advanced Report helps you build custom reports from your tracked events.

Use it when you want to:

- compare event trends over time
- analyze user movement between events
- apply filters and groups
- review charts and exact values together
- save a report for reuse later

<Note>
  Advanced Report has two main modes: **Segmentation** and **Funnel**.
</Note>

---

## Explore Advanced Report

<Tabs>
  <Tab title="Segmentation">
  ### Segmentation
    Segmentation helps you compare one or more events over time.

    It is useful when you want to understand how event activity changes by date, user group, or property.

    <Steps>
      <Step title="Open Segmentation">
        Start by opening **Advanced Report** and staying on the **Segmentation** tab.

        This is the starting point for trend analysis. From here, you can choose an app, date range, and begin building your report.

        <Frame>
          <img
            src="/images/Advance_report_1.png"
            alt="Advanced Report Segmentation starting view"
          />
        </Frame>
      </Step>

      <Step title="Add an event">
        Click **Add Event** and select the event you want to analyze.

        Each selected event appears as a block with a label such as **A**, **B**, or **C**. This helps you compare multiple events in the same report.

        <Frame>
          <img
            src="/images/Advance_report_2.PNG"
            alt="Advanced Report Segmentation with selected event"
          />
        </Frame>
      </Step>

      <Step title="Use Filter by and Group by">
        Use **Filter by** to narrow the report to a specific audience or condition.

        Use **Group by** to split the event into meaningful segments such as country, city, Android OS, brand, or other properties.

        <Frame>
          <img
            src="/images/Advance_report_3.PNG"
            alt="Advanced Report Segmentation with multiple events and group by"
          />
        </Frame>
      </Step>

      <Step title="Choose chart type and review results">

        You can switch the chart type based on how you want to view the data:

        - **Line Chart** for trends over time
        - **Bar Chart** for event comparison
        - **Stacked Bar** for grouped comparison
        - **Pie Chart** for distribution view

        <Frame>
          <img
            src="/images/Advance_report_4.png"
            alt="Advanced Report Segmentation chart type dropdown"
          />
        </Frame>
      </Step>

      <Step title="Save the report">
        After setting up your event selection, filters, chart type, and date range, click **Save**.

        Saving helps you keep the same report configuration so you or your team can open it from **Product Overview > reports**.

        <Frame>
          <img
            src="/images/advance_report_Common_save.png"
            alt="Advanced Report Segmentation save report step"
          />
        </Frame>
      </Step>
    </Steps>
  </Tab>

  <Tab title="Funnel">
### Funnel

    Funnel helps you measure how users move from one event step to the next.
    It is useful when you want to understand user conversion and identify where users drop off.

    <Steps>
      <Step title="Open Funnel">
        Switch from **Segmentation** to the **Funnel** tab.

        Funnel starts with step-based analysis. Instead of comparing event trends, it measures how many users completed each next step.

        <Frame>
          <img
            src="/images/Advance_report_funnel_1.PNG"
            alt="Advanced Report Funnel starting view"
          />
        </Frame>
      </Step>

      <Step title="Add the first event step">
        Select the first event in the funnel.

        The first event becomes the starting point of the conversion flow and is treated as **100%** of users entering the funnel.

        <Frame>
          <img
            src="/images/Advance_report_funnel_2.PNG"
            alt="Advanced Report Funnel with first event selected"
          />
        </Frame>
      </Step>

      <Step title="Add more funnel steps">
        Add the next events in the sequence to measure how users move forward.

        For example, you may start with **Fragment Viewed** and then add **Back Screen** as the second step.

        The chart and Funnel Table will then show how many users moved from the first step to the next one.

        <Frame>
          <img
            src="/images/Advance_report_funnel_3.PNG"
            alt="Advanced Report Funnel with multiple steps"
          />
        </Frame>
      </Step>

      <Step title="Apply filters or group the funnel">
        Use **Filter by** to narrow the funnel to a specific audience.

        Use **Group by** if you want to compare funnel conversion across segments such as country, state, city, Android OS, or brand.

        This helps you understand which user group converts better and where drop-off is higher.

        <Frame>
          <img
            src="/images/Advance_report_funnel_4.PNG"
            alt="Advanced Report Funnel with group by section"
          />
        </Frame>
      </Step>

      <Step title="Save the funnel report">
        Once your funnel steps, filters, and date range are set, click **Save**.

       Saving helps you keep the same report configuration so you or your team can open it from **Product Overview > reports**..

        <Frame>
          <img
            src="/images/advance_report_Common_save.png"
            alt="Advanced Report Funnel save report step"
          />
        </Frame>
      </Step>
    </Steps>
  </Tab>
</Tabs>

## What you can do in Advanced Report

<AccordionGroup>
  <Accordion title="Compare event trends">
    In Segmentation, you can compare one or more events across selected dates to understand whether activity increased, decreased, or stayed stable.
  </Accordion>

  <Accordion title="Analyze conversion">
    In Funnel, you can measure how many users completed the next event step and identify the exact drop-off between steps.
  </Accordion>

  <Accordion title="Use filters">
    Filters help you narrow the report to a specific user segment, property, or condition so that the output is more relevant.
  </Accordion>

  <Accordion title="Use group by">
    GroupBy lets you split the same event or funnel by country, city, OS, brand, and more, making comparison easier.
  </Accordion>

  <Accordion title="Save reusable reports">
    Save keeps the report configuration for future use so you do not need to rebuild the same analysis again.
  </Accordion>
</AccordionGroup>
