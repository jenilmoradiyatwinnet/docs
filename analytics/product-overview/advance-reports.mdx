# Advanced Report

Advanced Report helps you build custom analytics views without opening multiple dashboards.

Use it when you want to:
- compare one or more events over time
- break down data by filters or groups
- analyze conversion between steps
- switch between **Segmentation** and **Funnel**
- save a report for later use

<Note>
  Advanced Report has **two main tabs only** in the current UI: **Segmentation** and **Funnel**.
</Note>

### What Segmentation is

Segmentation helps you analyze how selected events perform across time.

You can compare:
- a single event
- multiple events
- filtered event groups
- grouped event breakdowns

It is useful when you want to answer questions like:
- How many times did an event happen?
- Which event is higher over time?
- Which country, user type, or property drives the event?
- How do two events compare in the same date range?

### When to use Segmentation

<AccordionGroup>
  <Accordion title="Compare multiple events">
    Use Segmentation when you want to compare events like <strong>Fragment Viewed</strong> and <strong>SplashTime</strong> in the same report.
  </Accordion>

  <Accordion title="Break down one event">
    Use <strong>Group by</strong> when you want one event split by country, city, OS, brand, or other properties.
  </Accordion>

  <Accordion title="Track trend over time">
    Use a line chart with a selected date range when you want to understand increase, drop, or pattern changes.
  </Accordion>

  <Accordion title="Export a detailed report">
    Use the Overview table when you need exact date-wise values and want to export them.
  </Accordion>
</AccordionGroup>

### What Funnel is

Funnel helps you measure how users move from one event to the next.

It is used to understand conversion flow, step drop-off, and step performance.

This is useful when you want answers like:
- How many users completed step 2 after step 1?
- Where are users dropping in a flow?
- What is the conversion rate from the first event?

### When to use Funnel

<AccordionGroup>
  <Accordion title="Measure drop between steps">
    Use Funnel when you want to know how many users moved from one event to the next.
  </Accordion>

  <Accordion title="Understand conversion rate">
    Funnel is useful when you need the percentage of users completing later steps from the starting event.
  </Accordion>

  <Accordion title="Compare conversion across segments">

  </Accordion>

## Suggested workflow

<Steps>
  <Step title="Choose the report type">
    Start by switching to either <strong>Segmentation</strong> or <strong>Funnel</strong> based on the analysis you want.
  </Step>

  <Step title="Set the date range">
    Select the time period first so all results are based on the correct window.
  </Step>

  <Step title="Add events">
    Add one or more events from the left-side panel.
  </Step>

  <Step title="Apply filters or groups">
    Narrow the report using <strong>Filter by</strong> or compare breakdowns using <strong>Group by</strong>.
  </Step>

  <Step title="Review chart and table">
    Use the chart for quick understanding and the table for exact values.
  </Step>

  <Step title="Save the report">
    Save the report if you want to reuse or share the same setup later.
  </Step>
</Steps>
