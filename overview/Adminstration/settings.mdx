Use this page to keep your workspace secure, organized, and up to date.

## How to use Settings

<Steps>
  <Step title="Open Settings">
    Go to Settings from the left sidebar.
  </Step>

  <Step title="Update profile and organization details">
    Use Manage Profile to update user information, address, organization logo, company details, and website.
  </Step>

  <Step title="Review security settings">
    Open Security & Settings to change password, set timezone, and manage two-factor authentication.
  </Step>

  <Step title="Check login history">
    Use Login History to review recent login activity by date, user, location, role, and time.
  </Step>

  <Step title="Invite team members">
    Go to Users and Permissions, enter email, select role type, and click Invite.
  </Step>

  <Step title="Manage permissions">
    Use the Permissions tab to control menu access for each role.
  </Step>
</Steps>
