Get started with Twinalyze in a few simple steps.\
This guide helps you create your account, log in, create your organization, and set up your first project for **Web** or **Android**.

  <Card title="Quick setup" icon="rocket">
    Create your account and get started with Twinalyze in a few minutes.
  </Card>

  <Card title="Web & Android" icon="monitor-smartphone">
    Follow a clear setup flow for both Web and Android projects.
  </Card>

  <Card title="Project ready" icon="folder-plus">
    Create your organization and project before starting SDK integration.
  </Card>
</CardGroup>

---

## Before you begin

Make sure you have:

- A valid email address
- Access to the Twinalyze dashboard
- Basic information about your app or website
- Your platform choice: **Web** or **Android**

<Info>
  You only need one Twinalyze account. After login, you can create multiple projects for Web, Android, Flutter, React Native, and other platforms.
</Info>

---

## Choose your platform

<Tabs>
  <Tab title="Web">
    ## Web Account Setup

    Follow these steps to create your account and start your first **Web** project.

    ### 1. Open the login page

    Visit the Twinalyze login page.

    <Frame>
      ![Twinalyze login page](/images/Login.PNG)
    </Frame>

    ### 2. Sign up with email and password

    If you do not already have an account, choose the sign-up option and enter your email and password.

    <Frame>
      ![Create account using email and password](/images/login_email_pass_page.PNG)
    </Frame>

    <Tip>
      Use a business email if you are setting up Twinalyze for your company or team.
    </Tip>

    ### 3. Log in to the dashboard

    After creating your account, log in and open the Twinalyze dashboard.

    <Frame>
      ![Logged in Twinalyze dashboard](/images/logged_in.PNG)
    </Frame>

    ### 4. Create your organization

    Your organization helps you manage projects, users, permissions, and settings in one place.

    <Frame>
      ![Create organization screen](/images/create-organization.PNG)
    </Frame>

    Recommended organization details:

    - Organization name
    - Company name
    - Product or brand name
    - Team details if needed

    ### 5. Create your Web project

    Now create your first Web project.

    <Frame>
      ![Create Web project screen](/images/create-project.png)
    </Frame>

    Recommended Web project details:

    - Project name
    - Platform: **Web**
    - Website name
    - Website URL
    - Time zone and region if needed
  </Tab>
  <Tab title="Android">
    ## Android Account Setup

    Follow these steps to create your account and start your first **Android** project.

    ### 1. Open the login page

    Open the Twinalyze dashboard and go to the login page.

    <Frame>
      ![Twinalyze login page](/images/Login.PNG)
    </Frame>

    ### 2. Register your account

    Create your account using your email and password.

    <Frame>
      ![Android account signup screen](/images/login_email_pass_page.PNG)
    </Frame>

    <Info>
      One account can manage multiple Android apps, Web projects, and other platform integrations.
    </Info>

    ### 3. Log in successfully

    After signup, log in to the Twinalyze dashboard.

    <Frame>
      ![Logged in Twinalyze dashboard](/images/logged_in.PNG)
    </Frame>

    ### 4. Create your organization

    Create an organization to manage your Android apps and analytics setup.

    <Frame>
      ![Create organization for Android project](/images/create-organization.PNG)
    </Frame>

    Recommended organization details:

    - Organization name
    - Company name
    - App or product name
    - Team information if needed

    ### 5. Create your Android project

    Create a new project and select **Android** as the platform.

    <Frame>
      ![Create Android project screen](/images/create-project.png)
    </Frame>

    Recommended Android project details:

    - Project name
    - Platform: **Android**
    - App name
    - Package name
    - Environment if needed

    ### 6. Continue with Android SDK setup

    After your Android project is ready, continue with SDK integration.

    <CardGroup cols={2}>
      <Card title="Android Kotlin SDK" icon="smartphone" href="/sdks/android/kotlin/analytics">
        Continue with Android Kotlin SDK installation and tracking.
      </Card>

      <Card title="Android Ad Analytics" icon="badge-ad" href="/sdks/android/kotlin/ad-analytics">
        Set up ad analytics tracking for your Android app.
      </Card>
    </CardGroup>
  </Tab>
</Tabs>

## Best practices

<AccordionGroup>
  <Accordion title="Use a clear organization name">
    Choose an organization name that matches your company, product, or brand. This helps your team manage projects easily.
  </Accordion>

  <Accordion title="Create separate projects for each platform">
    Keep Web and Android projects separate for better data clarity and reporting.
  </Accordion>

  <Accordion title="Check project settings before SDK integration">
    Make sure the platform, project name, and required details are correct before installing the SDK.
  </Accordion>

  <Accordion title="Use proper project naming">
    Use names like `Twinsho Web`, `Twinsho Android`, or `Client App Production` so your team can identify projects quickly.
  </Accordion>

  <Accordion title="Invite your team after setup">
    Once your setup is complete, invite teammates and assign roles from the dashboard.
  </Accordion>
</AccordionGroup>
