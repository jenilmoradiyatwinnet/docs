## Feature details

<AccordionGroup>
  <Accordion title="1. Real-Time Analytics" icon="chart-line">
    Real-Time Analytics shows what is happening inside your app or website right now.  
    It helps your team monitor live activity and react quickly when usage changes.

    | Capability | Simple explanation |
    | --- | --- |
    | Live Users | See how many users are active right now. |
    | Sessions | Track active and total user sessions. |
    | Screen Views | See which screens or pages users visit. |
    | Realtime Dashboards | Monitor key metrics instantly in one place. |

    <Tip>
      Best for live traffic monitoring, campaign impact, app usage, and sudden activity changes.
    </Tip>
  </Accordion>

  <Accordion title="2. User & Event Tracking" icon="users">
    User & Event Tracking helps you understand what users do inside your product.  
    It shows user actions, navigation flow, important events, and engagement behavior.

    | Capability | Simple explanation |
    | --- | --- |
    | Automatic Event Tracking | Capture important actions automatically. |
    | User Journeys | See the path users follow inside your product. |
    | Screen Tracking | Track app screens, web pages, and navigation flow. |
    | Custom Events | Track actions like signup, add to cart, checkout, payment failed, and purchase. |
    | Engagement Insights | Understand what users like and where they drop. |

    <Check>
      Use this to improve user flow, find drop-off points, and understand valuable actions.
    </Check>
  </Accordion>

  <Accordion title="3. Ads Analytics" icon="eye">
    Ads Analytics helps you measure how ads are performing inside your product.  
    It helps you track ad views, clicks, revenue, failed ad loads, and missed revenue opportunities.

    | Capability | Simple explanation |
    | --- | --- |
    | Impressions | Track how many times ads are shown. |
    | Clicks | Measure how many users click on ads. |
    | Revenue | Monitor ad earnings clearly. |
    | Failures | Detect failed ad loads and missed revenue. |
    | Ad Performance Optimization | Improve fill rate, clicks, revenue, and overall ad performance. |

    <Tip>
      Best for improving ad revenue, reducing ad failures, and finding missed monetization opportunities.
    </Tip>
  </Accordion>

  <Accordion title="4. Push Notifications & Automation" icon="bell">
    Push Notifications & Automation helps you send the right message to the right user at the right time.  
    It helps you create campaigns, automate messages, and track engagement.

    | Capability | Simple explanation |
    | --- | --- |
    | Scheduled Notifications | Send campaigns at a selected date and time. |
    | Multilingual Campaigns | Send messages in different user languages. |
    | Automated Messaging | Trigger messages based on user behavior or rules. |
    | Engagement Tracking | Track delivery, opens, clicks, and campaign response. |

    <Check>
      Use this to re-engage users, improve campaign performance, and increase retention.
    </Check>
  </Accordion>

  <Accordion title="5. Funnels, Retention & Reports" icon="filter">
    Funnels, Retention & Reports help your team understand conversion, repeat usage, and user segments.  
    It helps you find where users drop off and which users come back again.

    | Capability | Simple explanation |
    | --- | --- |
    | Conversion Funnels | Find where users convert and where they drop off. |
    | Retention Analytics | Measure how often users return after first use or key events. |
    | Cohort Analysis | Group users by date, event, behavior, or property. |
    | Custom Reports & Segmentation | Build reports using users, events, platforms, campaigns, countries, and filters. |

    <Tip>
      Best for finding drop-offs, measuring retention, comparing cohorts, and making better product decisions.
    </Tip>
  </Accordion>

  <Accordion title="6. Easy SDK Integration" icon="code">
    Easy SDK Integration helps developers add Twinalyze quickly.  
    After setup, Twinalyze can start collecting users, sessions, screens, events, and other important data.

    | Capability | Simple explanation |
    | --- | --- |
    | Lightweight SDK | Add tracking without heavy setup. |
    | Fast Setup | Install the SDK and start sending data quickly. |
    | Android/iOS Support | Track users across mobile platforms. |
    | Auto Data Collection | Collect important data automatically after setup. |

    <Check>
      Use this to reduce setup time and start tracking users faster.
    </Check>
  </Accordion>
</AccordionGroup>
