  <Step title="Analyze behavior">

## Core feature details

<Tabs>
  <Tab title="Analytics">
    ## Real-Time Analytics

    <Badge color="blue">Live product visibility</Badge>

    Real-Time Analytics helps you monitor what is happening inside your product right now.

    <CardGroup cols={2}>
      <Card title="Live Users" icon="radio">
        See how many users are active at the current moment.
      </Card>

      <Card title="Sessions" icon="clock">
        Track active sessions and understand product usage frequency.
      </Card>

      <Card title="Screen Views" icon="panel-top">
        Measure which screens or pages users are visiting.
      </Card>

      <Card title="Real-Time Dashboards" icon="gauge">
        Monitor key metrics instantly from one live dashboard.
      </Card>
    </CardGroup>

    <Tip>
      Best for monitoring live traffic, campaign impact, app usage, and sudden activity changes.
    </Tip>
  </Tab>

  <Tab title="Tracking">
    ## User & Event Tracking

    <Badge color="purple">Behavior intelligence</Badge>

    User & Event Tracking helps you understand what users do and how they move inside your product.

    <CardGroup cols={2}>
      <Card title="Automatic Event Tracking" icon="zap">
        Capture important actions automatically with SDK-based tracking.
      </Card>

      <Card title="User Journeys" icon="route">
        See the path users follow from entry to conversion.
      </Card>

      <Card title="Screen Tracking" icon="square-stack">
        Track app screens, web pages, and navigation behavior.
      </Card>

      <Card title="Custom Events" icon="code">
        Track actions like signup, add to cart, checkout, payment failed, or purchase.
      </Card>
    </CardGroup>

    <Info>
      Engagement insights help your team find what users like, where they drop, and which actions create value.
    </Info>
  </Tab>

  <Tab title="Ads">
    ## Ads Analytics

    <Badge color="green">Revenue performance</Badge>

    Ads Analytics helps you measure ad performance and find missed revenue opportunities.

    <CardGroup cols={2}>
      <Card title="Impressions" icon="eye">
        Track how many times ads are shown to users.
      </Card>

      <Card title="Clicks" icon="mouse-pointer-click">
        Measure how many users interact with ads.
      </Card>

      <Card title="Revenue" icon="banknote">
        Monitor ad earnings clearly across campaigns and placements.
      </Card>

      <Card title="Failures" icon="triangle-alert">
        Detect failed ad loads and revenue loss points.
      </Card>
    </CardGroup>

    <Tip>
      Use Ads Analytics to improve fill rate, reduce failures, and optimize ad revenue.
    </Tip>
  </Tab>

  <Tab title="Automation">
    ## Push Notifications & Automation

    <Badge color="orange">Engagement engine</Badge>

    Push Notifications & Automation helps you send the right message to the right user at the right time.

    <CardGroup cols={2}>
      <Card title="Scheduled Notifications" icon="calendar-clock">
        Schedule campaigns for a specific date and time.
      </Card>

      <Card title="Multilingual Campaigns" icon="languages">
        Send campaigns in different languages for different user groups.
      </Card>

      <Card title="Automated Messaging" icon="wand-sparkles">
        Trigger messages based on user behavior, events, or campaign rules.
      </Card>

      <Card title="Engagement Tracking" icon="chart-bar">
        Track delivery, opens, clicks, and campaign response.
      </Card>
    </CardGroup>

    <Check>
      Automation helps you re-engage inactive users, improve campaign performance, and increase retention.
    </Check>
  </Tab>

  <Tab title="Reports">
    ## Funnels, Retention & Reports

    <Badge color="pink">Growth insights</Badge>

    Reports help you understand conversion, repeat usage, cohorts, and segmented performance.

    <CardGroup cols={2}>
      <Card title="Conversion Funnels" icon="filter">
        Find where users convert and where they drop off.
      </Card>

      <Card title="Retention Analytics" icon="refresh-cw">
        Measure how often users return after first use or key events.
      </Card>

      <Card title="Cohort Analysis" icon="layers">
        Group users by date, event, behavior, or properties.
      </Card>

      <Card title="Custom Reports & Segmentation" icon="table-properties">
        Build reports using events, users, platform, campaign, country, and more.
      </Card>
    </CardGroup>

    <Tip>
      Use funnels and retention reports to make product decisions based on real user behavior.
    </Tip>
  </Tab>
