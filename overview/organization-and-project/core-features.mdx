## Core feature details

<Tabs>
  <Tab title="Analytics">
    ## Real-Time Analytics

    Real-Time Analytics helps your team see what is happening inside your product right now.

    <CardGroup cols={2}>
      <Card title="Live Users" icon="users">
        See how many users are active at the current moment.
      </Card>

      <Card title="Sessions" icon="clock">
        Track active and total user sessions.
      </Card>

      <Card title="Screen Views" icon="desktop">
        Understand which screens or pages users visit.
      </Card>

      <Card title="Real-Time Dashboards" icon="chart-line">
        Monitor key metrics instantly in one dashboard.
      </Card>
    </CardGroup>

    <Tip>
      Best for live traffic monitoring, campaign impact, app usage, and sudden activity changes.
    </Tip>
  </Tab>

  <Tab title="Tracking">
    ## User & Event Tracking

    User & Event Tracking helps you understand what users do and how they move inside your product.

    <CardGroup cols={2}>
      <Card title="Automatic Event Tracking" icon="zap">
        Capture important user actions automatically.
      </Card>

      <Card title="User Journeys" icon="route">
        See the path users follow from entry to conversion.
      </Card>

      <Card title="Screen Tracking" icon="desktop">
        Track app screens, web pages, and navigation flow.
      </Card>

      <Card title="Custom Events" icon="code">
        Track business actions like signup, add to cart, checkout, payment failed, or purchase.
      </Card>
    </CardGroup>

    <Check>
      Engagement insights help you find what users like, where they drop, and which actions create value.
    </Check>
  </Tab>

  <Tab title="Ads">
    ## Ads Analytics

    Ads Analytics helps you measure ad performance and find missed revenue opportunities.

    <CardGroup cols={2}>
      <Card title="Impressions" icon="eye">
        Track how many times ads are shown.
      </Card>

      <Card title="Clicks" icon="mouse-pointer">
        Measure how many users click on ads.
      </Card>

      <Card title="Revenue" icon="dollar-sign">
        Monitor ad earnings clearly.
      </Card>

      <Card title="Failures" icon="alert-triangle">
        Detect failed ad loads and missed revenue.
      </Card>
    </CardGroup>

    <Tip>
      Use Ads Analytics to improve fill rate, reduce failures, and optimize revenue.
    </Tip>
  </Tab>

  <Tab title="Automation">
    ## Push Notifications & Automation

    Push Notifications & Automation helps you send the right message to the right users at the right time.

    <CardGroup cols={2}>
      <Card title="Scheduled Notifications" icon="calendar">
        Schedule campaigns for a specific date and time.
      </Card>

      <Card title="Multilingual Campaigns" icon="language">
        Send campaigns in different languages.
      </Card>

      <Card title="Automated Messaging" icon="bell">
        Trigger messages based on user behavior or campaign rules.
      </Card>

      <Card title="Engagement Tracking" icon="chart-line">
        Track delivery, opens, clicks, and campaign response.
      </Card>
    </CardGroup>

    <Check>
      Automation helps re-engage inactive users and improve campaign performance.
    </Check>
  </Tab>

  <Tab title="Reports">
    ## Funnels, Retention & Reports

    Reports help you understand conversion, repeat usage, cohorts, and segmented performance.

    <CardGroup cols={2}>
      <Card title="Conversion Funnels" icon="filter">
        Find where users convert and where they drop off.
      </Card>

      <Card title="Retention Analytics" icon="rotate-cw">
        Measure how often users return after first use or key events.
      </Card>

      <Card title="Cohort Analysis" icon="layers">
        Group users by date, event, behavior, or property.
      </Card>

      <Card title="Custom Reports & Segmentation" icon="table">
        Build reports using users, events, platforms, campaigns, countries, and more.
      </Card>
    </CardGroup>

    <Tip>
      Use funnels and retention reports to make decisions based on real user behavior.
    </Tip>
  </Tab>
