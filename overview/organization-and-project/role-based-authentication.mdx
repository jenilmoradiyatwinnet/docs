## Give a role access

<Steps>
  <Step title="Choose a permission level">
    Open **Menu Group**, **Main Menu**, or **Sub Menu**.
  </Step>

  <Step title="Find the required feature">
    Use the filters or search field to find the menu or page.
  </Step>

  <Step title="Select the edit icon">
    Open the permission editor using the pencil icon.
  </Step>

  <Step title="Select allowed roles">
    Choose the roles that should be able to access the feature.
  </Step>

  <Step title="Confirm the permission is active">
    Keep the status toggle enabled and save the change.
  </Step>
</Steps>

---

## What do the controls mean?

| Control | What it does |
| --- | --- |
| **Role Type** | Shows permissions assigned to a selected role. |
| **Status** | Shows active or inactive permissions. |
| **Search** | Finds a menu or page by name. |
| **Pencil icon** | Opens the permission editor. |
| **Status toggle** | Enables or disables the permission. |

<Check>
  Start by inviting the user, assign the correct role, and then give that role
  access to the required Menu Groups and pages.
</Check>
