Twinalyze is an analytics and engagement platform that helps you understand how users interact with your app or website.

It tracks users, events, sessions, screens, sources, campaigns, notifications, and device data, and turns it into clear reports for product, marketing, and development teams.

## Why Twinalyze is useful

Twinalyze helps you answer key product questions in one place.

| Question | Insight |
| --- | --- |
| Who is using my product? | Users, devices, country, app version |
| What are users doing? | Events, clicks, screens, sessions |
| Where do users come from? | Sources, campaigns, UTM |
| Are users active? | Retention and engagement |
| Are campaigns working? | Notifications and conversions |

## Main data types

| Data type | Meaning | Example |
| --- | --- | --- |
| **User** | Person using your product | `user_1024` |
| **Event** | Action performed | `add_to_cart` |
| **Session** | One usage period | App used for 5 minutes |
| **Screen / Page** | User location | Home, product page |
| **Property** | Extra detail | price, country |
| **Source** | Traffic origin | Google Ads, organic |
