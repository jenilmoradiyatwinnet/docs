Twinalyze works by collecting user activity from your app or website and converting it into structured data that you can analyze through reports and dashboards.

## How it works

<Steps>
  <Step title="Connect">
    Add the Twinalyze SDK or API to your app, website, or backend.
  </Step>

  <Step title="Track">
    Capture user actions like clicks, screen views, sessions, and custom events.
  </Step>

  <Step title="Process">
    Data is organized into users, events, sessions, and properties.
  </Step>

  <Step title="Analyze">
    View dashboards, realtime users, reports, and insights.
  </Step>
</Steps>

## What gets tracked

| Type | Description |
|------|------------|
| **Users** | Who is using your product |
| **Events** | What actions they perform |
| **Sessions** | How long they stay |
| **Screens / Pages** | Where they go |
| **Devices** | Which device they use |
| **Sources** | Where they came from |

## Example flow

User opens app → views product → clicks button → completes action  

Twinalyze tracks each step and connects it to a user.

## Tracking methods

<CardGroup cols={2}>
  <Card title="Auto tracking" icon="bolt">
    Automatically tracks app opens, sessions, screen views, and basic activity.
  </Card>

  <Card title="Manual tracking" icon="pen">

  </Card>
</CardGroup>

## Why it matters

- Understand user behavior  
- Identify drop-offs and issues  
- Measure product and campaign performance  
- Make better product decisions
