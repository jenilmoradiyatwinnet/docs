## Getting started from dashboard

<Steps>
  <Step title="Create project">
    Enter your project name and click <strong>Save</strong>.
  </Step>

  <Step title="Open project settings">
    Configure SDK, API keys, and integrations.
  </Step>

  <Step title="Start tracking">
    Add SDK to your app or website.
  </Step>

  <Step title="View data">
    Start seeing users, events, and reports.
  </Step>
</Steps>
