## Events

Events are the foundation of analytics.

| Field | Meaning |
|------|--------|
| Event Name | Original event from SDK |
| Custom Name | User-friendly name |
| Source | Where event came from |
| Auto | Automatically tracked |
| Manual | Manually tracked |
| Visible | Shown in reports |
