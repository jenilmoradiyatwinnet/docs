## Quick map

| Dashboard section | Use it for |

| Ad | Ad report and ad failure analysis |
| Notification | Create campaigns and view notification reports |
| Research | Market trends and country/category insights |
| Data | Events and properties management |
| Project Setting | Project details, keys, versions, and configuration |
