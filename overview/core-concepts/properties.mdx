Twinalyze uses **properties** to add more context to your data.

> **Events = what happened**  
> **Users = who did it**  
> **Properties = extra details about it**

---

## What are Properties?

A **property** is additional information attached to users, events, or sessions.

Without properties:
- You only know → *purchase happened*

With properties:
- You know → *what was purchased, price, category, etc.*

---

## Example

```json
{
  "event": "purchase_completed",
  "userId": "user_1024",
  "properties": {
    "product": "Hair Clip",
    "price": 499,
    "category": "Accessories",
    "paymentMethod": "UPI"
  }
}
```
