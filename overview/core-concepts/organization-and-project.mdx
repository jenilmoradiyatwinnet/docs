## Organization vs Project

| Organization | Project |
|-------------|---------|
| Main workspace or company | App, website, or product |
| Contains team members | Contains analytics data |
| Can have multiple projects | Has its own API keys |
| Manages access and billing | Tracks users, events, sessions |

## Simple rule

> Organization is your company workspace. Project is the app or website where data is tracked.
