## How to Send an Ad Event

### Ad Request
   When an ad fetch is initiated.

<CodeGroup>

```kotlin Kotlin
Twinalyze.setAdsRequestEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String
    )
```

```java Java
Twinalyze.setAdsRequestEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String
    );
```

</CodeGroup>

### Ad Load
   When the ad successfully loads.
   
<CodeGroup>

```kotlin Kotlin
Twinalyze.setAdsLoadedEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String
    )
```

```java Java
Twinalyze.setAdsLoadedEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String
    );
```

</CodeGroup>


### Ad Show  
  When the ad is displayed.

<CodeGroup>

```kotlin Kotlin
Twinalyze.setAdsImpressionEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null
    )
```

```java Java
Twinalyze.setAdsImpressionEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null
    );
```

</CodeGroup>


</CodeGroup>

### Ad Failed 
   Captures load/show errors with error codes and messages.

<CodeGroup>

```kotlin Kotlin
Twinalyze.setAdFailedToLoadEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null,
        errorCode: String,
        errorMessage: String
    )
```

```java Java
Twinalyze.setAdFailedToLoadEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null,
        errorCode: String,
        errorMessage: String
    );
```

</CodeGroup>

### Ad Paid Event  
   Records revenue data only uses AdMob (micros, currency, precision).

<CodeGroup>

```kotlin Kotlin
Twinalyze.setAdsPaidEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null,
        currencyCode: String,
        precisionType: String,
        valueMicros: String
    )
```

```java Java
Twinalyze.setAdsPaidEvent(
        AdsCompany.Admob,
        AdsType.Interstitial,
        activityFullName: string,
        adUnitId: String,
        uniqId: String,
        bannerImpressionCount: String ?= null,
        currencyCode: String,
        precisionType: String,
        valueMicros: String
    );
```

</CodeGroup>
