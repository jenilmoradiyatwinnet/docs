Before tracking custom events, make sure the Twinalyze SDK is initialized.
## Track an event

Use the `track()` method and pass the event name as the first parameter.Pass an object as the second parameter to send additional information with the event.

<CodeGroup>

```javascript npm
TwinalyzeAnalytics.track("productViewed", 
 { //Add event propteries according to your needs
  productId: "PROD_101",
  productName: "Premium Plan",
  category: "Subscription",
  price: 999,
  currency: "INR",
})
```

```javascript CDN
window.TwinalyzeAnalytics.track("signUp",
 { //Add event propteries according to your needs
  productId: "PROD_101",
  productName: "Premium Plan",
  category: "Subscription",
  price: 999,
  currency: "INR",
})
```

</CodeGroup>

## Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `eventName` | String | Yes | Name of the custom event. |
| `properties` | Object | No | Additional information related to the event. |

## Examples

### Track product viewed

<CodeGroup>

```javascript npm
TwinalyzeAnalytics.track("ProductViewed", {
  productId: "email",
  productname: "Free",
  productcost:900,
});
```

```javascript CDN
window.TwinalyzeAnalytics.track("ProductViewed", {
   productId: "email",
  productname: "Free",
  productcost:900,
});
```

</CodeGroup>

### Track add to cart

<CodeGroup>

```javascript npm
TwinalyzeAnalytics.track("addToCart", {
  productId: "PROD_101",
  productName: "Analytics Plan",
  quantity: 1,
  price: 999,
  currency: "INR",
});
```

```javascript CDN
window.TwinalyzeAnalytics.track("addToCart", {
  productId: "PROD_101",
  productName: "Analytics Plan",
  quantity: 1,
  price: 999,
  currency: "INR",
});
```

</CodeGroup>

### Track a purchase

<CodeGroup>

```javascript npm
TwinalyzeAnalytics.track("purchaseCompleted", {
  orderId: "ORDER_1001",
  totalAmount: 1999,
  currency: "INR",
  paymentMethod: "card",
});
```

```javascript CDN
window.TwinalyzeAnalytics.track("purchaseCompleted", {
  orderId: "ORDER_1001",
  totalAmount: 1999,
  currency: "INR",
  paymentMethod: "card",
});
```

</CodeGroup>



## Recommended naming

Use clear and consistent event names that describe the completed user action.

<CodeGroup>

```javascript npm
TwinalyzeAnalytics.track("checkoutStarted");
TwinalyzeAnalytics.track("subscriptionPurchased");
TwinalyzeAnalytics.track("reportExported");
```

```javascript CDN
window.TwinalyzeAnalytics.track("checkoutStarted");
window.TwinalyzeAnalytics.track("subscriptionPurchased");
window.TwinalyzeAnalytics.track("reportExported");
```

</CodeGroup>
