Use `identify()` after a user logs in, registers, or when their profile information becomes available.

The method connects anonymous website activity with a known user.
<Note>
For npm installations, use `TwinalyzeAnalytics`.

For CDN installations, use `window.TwinalyzeAnalytics`.
</Note>
## Identify a user

## Identify with user properties

You can send additional user information as the second parameter.

<Tabs>

    ```javascript
      email: "john@example.com",
      phone: "+919876543210",
      plan: "Growth",
      company: "Example Company",
    });
    ```

  </Tab>

  <Tab title="CDN">

    ```html
        name: "John Doe",
        email: "john@example.com",
        phone: "+919876543210",
        plan: "Growth",
        company: "Example Company",
      });
    </script>
    ```
  </Tab>
</Tabs>

</div>

## When to call identify

Call `identify()` when:

- A user successfully logs in
- A user completes registration
- User profile information becomes available
- User properties such as plan or company change

```javascript
function handleLogin(user) {
  TwinalyzeAnalytics.identify(user.id, {
    name: user.name,
    email: user.email,
    plan: user.plan,
  });
}
```

## Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `userId` | String | Yes | Unique identifier of the user. |
| `identifyProperties` | Object | No | User profile information such as name, email, phone, plan, or company. |

<Info>
Initialize the Twinalyze SDK before calling `identify()`.
</Info>

<Warning>
Use a permanent and unique user ID from your application. Do not use values that may change, such as a display name.
</Warning>
