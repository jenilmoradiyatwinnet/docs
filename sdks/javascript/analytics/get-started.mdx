Welcome to the **Twinalyze JavaScript Analytics SDK**, your gateway to real-time insights for websites and web applications.

Twinalyze automatically captures important website interactions after the JavaScript SDK is initialized.

---

## What’s Inside?

| SDK | Description |
| --- | --- |
| **JavaScript Analytics SDK** | Tracks website visitors, sessions, page views, clicks, forms, searches, downloads, custom events, and identified users. |

<Info>
Initialize the SDK once and Twinalyze will automatically start tracking supported website activity.
</Info>



## What You’ll Learn

- How to install and initialize the JavaScript SDK
- What events are tracked automatically
- How to track custom events
- How to identify logged-in users
- How to manage automatic event tracking
