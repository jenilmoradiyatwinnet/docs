## What it will track

<CardGroup cols={2}>
  <Card title="Page views">
    Track which pages users visit, including page URL, path, title, domain, and referrer details.
  </Card>

  <Card title="Custom events">
    Send important user actions such as button clicks, form submissions, signups, purchases, or any custom website interaction.
  </Card>

  <Card title="Session activity">
    Understand how users move through your website and how long they stay active.
  </Card>

  <Card title="Traffic source">
    Capture referrer, UTM, source, medium, campaign, and other acquisition details.
  </Card>
</CardGroup>

---

## What will be available

<Steps>
  <Step title="Install the JavaScript SDK">
    Add the Twinalyze JavaScript SDK to your website or web application.
  </Step>

  <Step title="Initialize the SDK">
    Connect your website with your Twinalyze project using the project credentials.
  </Step>

  <Step title="Track automatic events">
    Capture page views and basic web behavior with minimum setup.
  </Step>

  <Step title="Send custom events">
    Track business-specific actions with event names and custom properties.
  </Step>
</Steps>

## Preparation checklist

<CardGroup cols={2}>
  <Card title="Keep project credentials ready">
    You will need your Twinalyze API key and project details before starting the setup.
  </Card>

  <Card title="Review website events">
    Decide which user actions should be tracked as custom events.
  </Card>

  <Card title="Plan event properties">
    Prepare useful properties such as page name, button name, product ID, amount, currency, or user type.
  </Card>

  <Card title="Check traffic tracking">
    Make sure your website uses proper URLs and UTM parameters for campaign analysis.
  </Card>
</CardGroup>
