No additional `track()` calls are required.

## Event details

<AccordionGroup>

  <Accordion title="Page views ‚Äî pageView">

    Captured when a visitor:

    - Opens a page
    - Uses browser back or forward
    - Changes routes in a single-page application

    **Properties**

    `pageTitle`, `pageDomain`, `pageUrl`, `pagePath`, `previousPageUrl`, `previousPageType`

  </Accordion>

  <Accordion title="Element clicks ‚Äî elementClick">

    Captured when a visitor clicks a supported link, button, menu item, dropdown option, or interactive element.

    **Properties**

    `clickType`, `elementTag`, `elementText`, `elementId`, `elementName`, `elementRole`

    Link clicks can also include:

    `linkUrl`, `linkDomain`, `linkPath`, `isExternalLink`

    Button clicks can also include:

    `buttonType`, `disabled`

  </Accordion>

  <Accordion title="Scroll depth ‚Äî scrollDepth">

    Captured when a visitor reaches at least 90% of a page.

    **Property**

    `scrollPercent`

    The event is captured once for each page view.

  </Accordion>

  <Accordion title="Form start ‚Äî formStart">

    Captured when a visitor first interacts with a form.

    **Properties**

    `formId`, `formName`, `formActionUrl`

    The event is captured once for each form during the current page visit.

  </Accordion>

  <Accordion title="Form submit ‚Äî formSubmit">

    Captured when the browser submits a form.

    **Properties**

    `formStatus`, `formId`, `formName`, `formActionUrl`, `formDestinationUrl`

    <Note>
    This event records the browser submission. It does not confirm that the server successfully processed the form.
    </Note>

  </Accordion>

  <Accordion title="Site search ‚Äî searchResultsView">

    Captured when a search term is found in a supported URL parameter.

    **Supported parameters**

    `q`, `s`, `search`, `query`

    **Example**

    ```text
    https://example.com/search?q=analytics
    ```

    **Properties**

    `searchKey`, `searchTerm`, `searchUrl`

  </Accordion>

  <Accordion title="File downloads ‚Äî fileDownload">

    Captured when a visitor clicks a supported file link.

    **Supported files**

    `pdf`, `zip`, `apk`, `doc`, `docx`, `xls`, `xlsx`, `ppt`, `pptx`

    **Properties**

    `linkUrl`, `fileExtension`, `fileName`

  </Accordion>

</AccordionGroup>
